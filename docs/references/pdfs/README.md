# Reference PDFs

Store local PDFs in numbered topic subdirectories:

```text
NN-<topic>/
```

Use the registry ID as the filename prefix:

```text
REF-NNN-<short>.pdf
```

Example:

```text
01-redox-fo2/REF-001-kress-carmichael-1991.pdf
```

The PDF archive mirrors the Propulsion `reference-library-pdfs/` convention.
The YAML registry remains the source of truth; PDFs are supporting evidence.
